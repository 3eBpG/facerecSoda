# facerecSoda
Суть проекта заключена в том, что у нас есть смежный проект где ребята делают автомат с газировкой, который реагирует на оплату по qr коду и выдает напиток
Мы в свою очередь добавляем к их автомату функцию распознавания студентов по их лицу, чтобы узнать какая у покупателя успеваемость, и за хорошие оценки выдавать бесплатные напитки.
Для реализации сначала были использованы OpenCV и numpy чтобы обучить нейросеть на базе данных студентов из группы. 
В дальнейшем при попытке настроить распознавние чтобы оно было более четким, была обнаружена библиотека face_recognition в которую нам надо уже имеющийся датасет запихать для присвоения лицам имен и id. Чтобы оно понимало кто именно перед ней.
Также была создана база данных на postgreSQL к которой мы и хотим обращаться из нашей программы для определения успеваемости студента
